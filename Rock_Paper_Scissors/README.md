# Rock-Paper-Scissors Game

## Instruction:

This is a simple Rock-Paper-Scissors Game game GUI built by Python and Tkinter.

The application is distributed by PyInstaller.

## How to run:

have python3 (tkinter) installed

```Shell
python3 game.py
```
